# 📊 Student Attendance Data Visualizer

An interactive MATLAB dashboard for analyzing and visualizing student attendance data across schools.

---

## 📂 Dataset

**2018-2019 Daily Attendance** — `2018-2019_Daily_Attendance_20240429.csv`

Daily attendance records per school containing: School DBN, Date, Enrolled, Present, and Absent counts.

---

## 🛠️ Tools
`MATLAB` · `Statistics and Machine Learning Toolbox`

---

## ✨ Features

| Control | Options |
|---|---|
| School Selector | All Schools or any individual school |
| Plot Types | Time Series, Bar Chart, Scatter Plot, Box Plot, Histogram, 3D Surface |
| Metrics | Attendance Rate, Absence Rate, Enrolled, Present, Absent |
| Trend Panel | Moving average overlay on daily data |
| Statistics | Live stats panel + detailed popup window |

---

## 🚀 How to Run

1. Place `Attendance.m` and the CSV file in the same folder
2. Open `Attendance.m` in MATLAB and run
3. Select the CSV file when prompted
4. Use the dropdowns to explore different schools, plot types, and metrics
